Welcome to Match-Match Game!

About This Game:

Match-Match Game is a memory game where the object is to find pairs.

When the game begins all the pictures are hidden.

How to Play:

  1. Select two cards to try to match the pictures.
  2. If you match the pictures they will disapear and you can go again.
  3. If they don't match the computer turn them.
  4. If you match all the pictures you are the winner!
  5. Have fun =)
